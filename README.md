# SpeedLimit
UK speed limit notifier.



## Adding to home screen

### Android

Open [the page](skambaman.github.io/SpeedLimit) in Chrome.

Tap the Three Dots Menu (â‹®) in the top-right corner.

Select "Add to Home screen".

You can rename it (e.g., to "Car Speed").

Tap Add.


### iOS

Open [the page](skambaman.github.io/SpeedLimit) in Safari.

Tap the "Share" button (square with arrow).

Select "Add to Home Screen".

Launch it from the home screen icon like a real app. This gives it better permissions to keep the screen awake.
